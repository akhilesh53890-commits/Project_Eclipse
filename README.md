# Project Eclipse

A psychological horror game developed in Unity.

## Directory Structure

```
Project_Eclipse/
│
├── Documentation/        # Game design documents, story outlines, character profiles, tasks
│   ├── Core_Idea.docx / .md
│   ├── Game_Design_Document.docx / .md
│   ├── Story.docx / .md
│   ├── Characters.docx / .md
│   ├── Gameplay.docx / .md
│   ├── Tasks.xlsx / .md
│   └── References/       # Concept art references, audio samples, research notes
│
├── Assets/               # Shared raw game assets (3D models, textures, audio, fonts)
│
├── UnityProject/         # Unity Engine Project root directory
│
├── Concepts/             # Concept sketches, storyboards, moodboards
│
├── UI/                   # UI mockups, icons, HUD vector assets
│
└── Builds/               # Compiled executable builds (Dev, Alpha, Beta, Release)
```

## Quick Start
- Refer to `Documentation/Game_Design_Document.md` for overall vision.
- Refer to `Documentation/Tasks.md` (or `Tasks.xlsx`) for task tracking.
- Open `UnityProject/` in Unity 2022.3 LTS or higher.
